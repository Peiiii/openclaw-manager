# OpenClaw Manager 规划（v0.1 目标）

## 产品定位

- 面向本机/VM/VPS 的 OpenClaw 全流程管理入口
- 以最小输入完成安装、配置、验证、运行
- 可扩展到多机管理、远程部署与集群化控制

## 核心体验里程碑

1. 本地一键启动 + Web 引导（已完成）
2. CLI 自动安装与版本检查
3. 网关启动、频道探测、配对审批闭环
4. 诊断/修复（端口占用、配置缺失、token 失效）
5. 远程模式（SSH 安装/执行）
6. 多节点视图（本机 + VPS + 家庭机）

## 技术架构

- `apps/web`：前端 UI（轻逻辑 + 状态展示）
- `apps/api`：本地控制平面（命令执行 + 进程管理 + 状态聚合）
- `packages/core`：工作流与步骤状态机（流程声明式定义）
- `packages/shared`：类型、协议与校验
- `packages/cli`：一键启动 + 诊断工具

## 验收标准

- 新用户只需 1 条命令启动 Web 界面
- 核心链路可通过 UI 完成（token、配对、探测）
- 所有关键节点可通过 API 自动验证
- Cloudflare Pages 一键部署成功，并可访问首页
